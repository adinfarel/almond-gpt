This project is a hands-on exploration of building a GPT model from scratch. It implements custom Byte Pair Encoding (BPE) tokenization and a complete Transformer decoder architecture, trained on the Dolly-15K dataset for educational purposes.

Technology Stack

Infrastructure & MLOps

A production-grade architecture built for scalability, monitoring, and reproducibility.

Docker

Containerization platform that packages the application and all dependencies into isolated containers, ensuring consistent deployment across different environments.

FastAPI

Modern, high-performance Python web framework for building APIs. Handles HTTP requests, streaming responses, and serves the model inference endpoints.

Prometheus

Open-source monitoring and alerting toolkit. Collects and stores metrics as time-series data, enabling real-time monitoring of model performance and system health.

MLflow

Platform for managing the ML lifecycle, including experiment tracking, model versioning, and deployment. Tracks hyperparameters, metrics, and model artifacts.

Grafana

Analytics and interactive visualization platform. Creates dashboards to visualize metrics from Prometheus, providing insights into model behavior and system performance.

DVC

Data Version Control for ML projects. Manages large datasets and model files, enabling versioning, sharing, and reproducibility of data pipelines.
